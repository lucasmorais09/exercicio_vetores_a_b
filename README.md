# exercicio_vetores_a_b
Recebe valores para os vetores A e B, faz a multiplicação entre eles e exibi o resultado.
------------------------------------------------------
# ALGORITMO DO EXERCÍCIO
Início 

	  Declarar vetores A[] e B[]
	  Declarar variável C
	  Enquanto x < 5. Faça 
	    Receber e atribuir valores de A e B
	    Somar x + 1
	  FimEnquanto
	  Enquanto x < 5. Faça
	    c = c + A[x] * B[x]
	  FimEnquanto
	  Exibir valor de c
  
Fim
